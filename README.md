# 📝 To-Do List Web App

A simple and interactive **To-Do List application** built using **HTML, CSS, and JavaScript**. This project allows users to manage their daily tasks efficiently by adding, editing, completing, and deleting tasks.

---

## 🚀 Features

* ➕ Add new tasks
* ⌨️ Press **Enter** to quickly add a task
* ✔ Mark tasks as **completed**
* ✏ **Edit** existing tasks
* 🗑 **Delete** tasks
* 🔢 **Task counter** to track total tasks
* 🎨 Clean and responsive user interface

---

## 🛠 Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Task management logic and DOM manipulation
